# 🧩 Puzzle

A jigsaw puzzle web app that recreates the experience of laying a physical puzzle on a table.

Pieces merge with each other when matching edges align — no snapping to a fixed grid. Just like a real puzzle.

## Development

```bash
npm install
npm run dev
```
